# Proyecto_integrador2
Códigos y explicaciones para el desarrollo del proyecto integrador 2.  Inhibición del gen de la proteina CCR9

# Instalar local
1. Descargar todo.
2. No directorio base (donde están backend, frontend e compose) con docker aberto executar en cmd/powershel "docker compose build" (para que vaia un pouco máis rápido añadide ao final --parallel).
3. Executamos "docker compose up" para que se execute todo.

En caso de facer cambios no código repetir pasos 2 e 3.

Se dá erros comprobade que non teñades contenedores que estan ocupando os puertos 3000, 5001, 5002 e 8000.

Para xogar coa interfaz localhost:8000
